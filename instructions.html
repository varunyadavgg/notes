Contribution Instructions

Welcome to the Subject Notes contribution instructions blog! We're delighted that you're interested in contributing to our project. Follow the steps below to make your contribution.

Step 1: Fork the Repository

Fork the Subject Notes repository on GitHub by clicking on the "Fork" button at the top-right corner of the repository page.

Step 2: Clone the Repository

Clone your forked repository to your local machine using the following command:

git clone https://github.com/yourusername/subject-notes.git

Step 3: Make Changes

Edit the notes.json file to add or modify notes. You can also make changes to the CSS files to improve the styling.

Step 4: Commit and Push Changes

Commit your changes and push them to your forked repository:

git add .
git commit -m "Add your commit message here"
git push origin master

Step 5: Create a Pull Request

Go to the GitHub page of your forked repository and click on the "New Pull Request" button. Provide a brief description of your changes and submit the pull request.

Step 6: Review and Merge

Wait for the project maintainers to review your pull request. Once approved, your changes will be merged into the main repository.

Thank you for contributing to Subject Notes!
